# Fiiiilm

## Getting Started

1. You need to set [fiiiiilm-backend](https://github.com/engulfedInFlames/fiiiiilm-backend) up.
2. Install `node_modules`.

```zsh
npm i
```

3. Run the project.

```zsh
npm start
```
